This is reusable animation. 
С помощью метода animation() описывается анимация. В двойных фигурных скобках описаны параметры, передаваемые анимации при ее вызове через useAnimation().
Также допустимо использование константных значений.

Функция useAnimation() принимает два параметра:
  1) анимация - определення для переиспользования;
  2) объект - в свойстве params которого указываються значения параметров.
